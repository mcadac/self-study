Formularios

  1. Las etiquetlas de los formularios se dividen en 2 tipos, captura de datos y enunciado
  2. Etiquetas
    • fieldset : Envuelve todo el contenido despues de la etiqueta form
    • legend : Establece un titulo al formulario (ubicado en el fieldset)
    • input :
      • Tiene un atributo type que ofrece diversas opciones de tipos de datos.
      • atributo max y min = aplica para tipos numericos, indicando el valor maximo o minimo que puede tomar
      • atributo step = aplica para tipos num, indica de a cuanto se realizara el incremento o decremento del campo e.g 5 en 5 (10,15,20)
    • select : Campo de selección de opciones con una etiqueta interna "option"
    • textarea : tiene 2 atributos rows y cols para ajustar el tamaño
    • label : tiene un atributo for que relaciona con el id de un etiqueta input
